## Overview

Growth and resilience are not the same thing.

Most destination reporting focuses on visitor arrivals, total expenditure and year-on-year growth. Those measures describe how demand has changed. They reveal much less about what that demand depends on.

This publication introduces a portfolio diagnostic that compares each Australian state's visitor expenditure mix against structurally comparable destination types. Rather than asking whether demand is growing, it asks whether the underlying portfolio is balanced, concentrated or exposed.

The publication is accompanied by an interactive diagnostic that allows every state and visitor purpose to be explored.

## Method (summary)

The publication compares visitor expenditure shares within structurally comparable destination types using expenditure-weighted benchmarks.

Destination types used in the analysis are:

- Urban Gateway
- Coastal Leisure
- Regional Mixed
- Remote

Differences of ±0.5 percentage points or less are treated as broadly at benchmark.

The Northern Territory is treated as a special case because it has no peer destination within the dataset against which portfolio concentration can be benchmarked.

The analysis is based on one year of publicly available data and is intended as a directional diagnostic to support discussion and further investigation. It should not be interpreted as an official Tourism Research Australia publication, definitive forecast or policy recommendation.

A full explanation of the analytical framework is provided within the publication and the accompanying diagnostic repository.
